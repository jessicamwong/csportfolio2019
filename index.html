Intro to Programming

Throughout my Intro to Programming class, I am learning how to code using HTML, CSS, and Java Script. Since I've never been exposed to coding, all of this information is new to me. The first thing that I learned was the HTML basics. This covered a variety of tags such as the following:

  • body
  • p
  • h1, h2, h3, h4, h5, h6
  • br
  • ol/ul
  • em
  • img
  • table
  • tr
  • td

These tags allow me to create a multitide of things, including web pages, such as the one you are looking at right now. Before I took the class, I always thought that coding was boring. However, I soon learned that coding was actually really fun and cool. I never knew how much you can do when you just play around with the code. Though, coding can be quite hard and frustrating at times. Sometimes I would forget to add an end tag or maybe mispell a tag and it would take me a really long time to figure out why certain content wasn't appearing on my web page. I would stare at my code for hours or try to edit parts of it to see if it would change anything. This process would repeat until I figured out what was wrong. Through this process, I learned patience and precision. Patience is important when coding because it can be a very tedious process at times. Precision is also super important because one small mistake, such as forgetting to add a semi colon or mispelling a tag, can mess up your whole code.
